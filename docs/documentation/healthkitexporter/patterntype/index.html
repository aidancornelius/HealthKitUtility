Enumeration

PatternType

Pattern transformation types for generating synthetic health data.

Declaration

enum PatternType

Overview

These patterns modify existing health data to create variations for testing. Stress-related patterns affect both heart rate and HRV in physiologically appropriate ways.

Topics

Enumeration Cases

case amplified
Increase stress indicators (higher HR, lower HRV)
case inverted
Flip high and low stress periods
case random
Add significant random variations
case reduced
Decrease stress indicators (lower HR, higher HRV)
case similar
Keep similar patterns with minor random variations
