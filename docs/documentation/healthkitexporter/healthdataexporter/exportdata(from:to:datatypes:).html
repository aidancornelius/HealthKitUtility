Instance Method

exportData(from:to:dataTypes:)

Exports health data from HealthKit for a specified date range.

Declaration

@MainActor func exportData(from startDate: Date, to endDate: Date, dataTypes: Set<HealthDataType>) async throws -> ExportedHealthBundle

Parameters

startDate

The start of the date range to export

endDate

The end of the date range to export

dataTypes

Which health data types to include in the export

Return Value

A bundle containing all exported health data

Discussion

This method queries HealthKit for all requested data types within the date range and returns them as a structured bundle ready for JSON serialisation.

Note

Various HealthKit errors if data cannot be accessed
